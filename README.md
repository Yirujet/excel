## 简介

tiny-excel 是一个由 canvas 实现的简易版高性能网页 Excel，由原生 js 实现，跨框架，在 Angular、React、VUE 中均可正常使用。

```
// 安装
npm i tiny-excel

// 使用
import Excel from 'tiny-excel'
```
